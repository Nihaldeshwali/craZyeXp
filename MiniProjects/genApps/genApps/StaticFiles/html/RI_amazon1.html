Is your array have any Duplicates ?

You are given an integer array. You need to find out if the array contains any duplicates.

Work out some examples !

How complex the problem is, excesing the problem with example helps a lot to understand the problem better! To make this problem more clear let consider the follwoing example.
  • Example 1 : You are given a below array {1,2,3,4,5}. All the element in the array occur only once. hence the result will be "No Duplicates"
  • Example 2: Now consider this array {1,2,3,4,5,6,7}. In this arry element 1 is occure twice at the begining and at the end , hence it will say "Duplicate Found"

For this Problem, it also have a native way to solve this.Here is how:
  • Scan the array from left to right and pick-up an element at a time.
  • Compare this item with other elements to verify whether they are same or not!
    • If we found they are same!, we know the duplicate exist for that elemnt - hence print "Duplicate Found".
  • When we complete the processing for all the elements and no such dupliacte elements found, we can certainly conclude that the array contains no dupliacte.

Derive the Complexity!

What is the time complexity of this problem ? O(n^2) - This is because we picked up each element and compare with all the elements in right side of this. thus For first element we need N comparism , For second elemnt it need n-1 comparisn and so on. Thus total comparrism require is N+(N-1)+(N-2)+...+1 = O(n^2)
What is the space complexity for this problem? It's O(1), As we are not using any space other than loop veraibles ( Which takes constact space), which depends on input size.
